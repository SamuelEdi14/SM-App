# Aplikasi Silap Mata

Aplikasi Silap Mata adalah sebuah *prototype* yang bertujuan untuk melaporkan pelanggaran - pelanggaran rambu lalu lintas dari masyarakat sekitar kepada pihak berwajib khususnya di daerah Pekanbaru, Riau.
